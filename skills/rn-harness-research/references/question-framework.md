# Question Framework for Market Research

시장 조사 시 체계적으로 답해야 할 질문 프레임워크.

## Phase 1: 시장 이해

1. 이 카테고리의 전체 시장 규모는?
2. 최근 1년간 성장 추세는?
3. 주요 플레이어는 누구인가?
4. 사용자들이 가장 많이 하는 불만은?
5. 미충족 니즈는 무엇인가?

## Phase 2: 경쟁 분석

1. 상위 앱들의 공통 기능은?
2. 차별화된 기능을 가진 앱은?
3. 평점이 낮은 앱의 주요 문제점은?
4. 최근 출시된 신규 앱의 접근법은?
5. 유료/무료 앱 비율은?

## Phase 3: 사용자 분석

1. 타겟 사용자의 연령/성별/직업은?
2. 사용자가 이 앱을 쓰는 상황(context)은?
3. 사용자의 주요 페인 포인트는?
4. 대안으로 현재 무엇을 사용하는가?
5. 얼마나 자주 사용할 것인가?

## Phase 4: 수익화

1. 이 카테고리에서 가장 성공적인 수익 모델은?
2. 사용자의 지불 의향은?
3. 광고 기반 수익화가 적절한가?
4. 프리미엄/구독 모델이 가능한가?
5. AdMob 예상 eCPM은?
